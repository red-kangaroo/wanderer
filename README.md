# Wanderer
</div>
<br/>
<br/>
<br/>
A small roguelike game written in JavaScript.

### Controls

```
Move around with
                 W
 arrow keys    A   D
                 S

     8           K
   4   6       H   L
     2           J

Use Space, '.' or '5' to wait a turn.

Use 'X' or '0' to invoke the powers of your magic staff, if you have one.

Press '?' in game for more help.
```

Have fun!
